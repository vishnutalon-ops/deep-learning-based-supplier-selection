# Deep Learning-Based Supplier Selection

## Overview

This project presents a deep learning-driven supplier selection framework using sentiment analysis and multi-criteria decision-making (MCDM).

## Methodology

### Phase 1: Sentiment Analysis

* VADER model
* RoBERTa model (fine-tuned)
* Model comparison

### Phase 2: Decision Framework

* Criteria extraction
* Weight calculation
* Sensitivity analysis
